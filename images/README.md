# Afbeeldingen — exacte bestandsnamen

Zet je foto's hier met **exact** deze namen. Dan verschijnen ze automatisch op de juiste plek;
je hoeft niets in de code te wijzigen. De vakjes hebben een vaste verhouding, dus de lay-out
verschuift niet als je een foto vervangt.

| Bestandsnaam            | Wat                                   | Ideale verhouding | Waar op de site            |
|-------------------------|----------------------------------------|-------------------|----------------------------|
| `headshot.jpg`          | Jouw pasfoto (boarding pass)           | 4:5 (staand)      | Leg 0 — hero               |
| `logo-ide-coffee.png`   | Logo IDE Coffee (liefst transparant)   | vrij (wordt passend getoond) | Leg 2 — stempel  |
| `logo-jaimy.png`        | Logo Jaimy by Belfius (transparant)    | vrij (wordt passend getoond) | Leg 2 — stempel  |
| `bruzelle-cover.png`    | Cover infoboekje BruZelle              | 1:1 (vierkant)    | Leg 3 — cargo              |

Optioneel (nu niet gebruikt, maar kan toegevoegd worden):
- `logo-odisee.png` · `logo-nationale-loterij.png` · `jaimy-blog-grid.png` · `jaimy-website.png`

**Tips**
- Houd bestanden klein (< 250 KB) voor een snelle site. JPG voor foto's, PNG voor logo's met transparantie.
- Een vervangen foto verschijnt na een **harde verversing** in je browser (Ctrl + F5).
